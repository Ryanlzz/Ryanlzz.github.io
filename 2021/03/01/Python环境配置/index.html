Anaconda+CDUA+CUDNN+Pytorch环境配置

1.下载安装Anaconda

下载地址: https://www.anaconda.com/products/individual

2.下载安装Pycharm

下载地址: https://www.jetbrains.com/pycharm/download/#section=win

安装激活: http://lookdiv.com

3.下载安装CUDA和cudnn

我安装的都是最新版的cuda11.1和cudnn8.0.5，安装TensorFlow的时候才发现有坑……。

CUDA: https://developer.nvidia.com/cuda-toolkit-archivE

cudnn: https://developer.nvidia.com/rdp/cudnn-download

4.创建虚拟环境

为什么创建虚拟环境?

虚拟环境其实就是Python环境的一个副本，可以保证每个项目都有一套独立的python运行环境，在不同的虚拟环境中可以选择不同版本的库。

创建虚拟环境，主要用于TensorFlow

  1. 创建虚拟环境：conda create —name tensorflow-gpu python=3.8，创建在Anaconda3的envs下。

  2. 激活环境：conda activate tensorflow-gpu

  3. 查看环境：conda info —envs

  4. 安装TensorFlow：pip install tensorflow-gpu

  5. 打开python，输入 import tensorflow as tf 报如下错误：

    Could not load dynamic library ‘cudart64_101.dll’; dlerror: cudart64_101.dll not found

    原因是我装的TensorFlow版本是最新的tensorflow-gpu 2.3.1，但是它还不支持CUDA11.1，办法是下载低版本的CUDA和cudnn。

    tensorflow各个版本需要的CUDA版本以及Cudnn的对应关系：https://www.tensorflow.org/install/source_windows#install_gpu_support_optional，

    之后找到解决办法参考：https://blog.csdn.net/qq_32939413/article/details/105525025

    不过我也不着急用，再等等TensorFlow更新吧lol，听说Pytorch支持DUDA11，那用Pytorch吧。

  6. 退出环境：deactivate

参考：

  1. https://zhuanlan.zhihu.com/p/60647332
  2. https://blog.csdn.net/j879159541/article/details/93199150

创建虚拟环境，主要用于Pytorch

  1. 创建虚拟环境：conda create —name pytorch python=3.8

  2. 激活环境：conda activate pytorch

  3. 查看环境：conda info —envs

  4. 安装Pytorch：conda install pytorch torchvision torchaudio cudatoolkit=11.0 -c pytorch，参考官网：https://pytorch.org/，要等一段时间了......。

  5. 打开python，输入 import torch，无报错，成功！

  6. 安装pandas：conda install pandas

  7. 安装Matplotlib：conda install matplotlib

  8. 安装scikit-image：conda install scikit-image

  9. 安装sklearn：conda install scikit-learn

  10. 安装opencv-python：pip install opencv-python -i https://pypi.mirrors.ustc.edu.cn/simple

    下载太慢了，我使用了中科大的源下载：

    要换哪个源只需要改-i后面的链接地址即可

    阿里云 http://mirrors.aliyun.com/pypi/simple/

    中国科技大学 https://pypi.mirrors.ustc.edu.cn/simple/

    豆瓣(douban) http://pypi.douban.com/simple/

    清华大学 https://pypi.tuna.tsinghua.edu.cn/simple/

    中国科学技术大学 http://pypi.mirrors.ustc.edu.cn/simple/

  11. 安装jupyter notebook：conda install jupyter notebook

注：如果安装中途安装失败，可以使用：conda clean —all，然后重新安装，不然可能导致CondaVerificationError

创建虚拟环境，用于labelme

  1. 创建虚拟环境：conda create —name=labelme python=3.7
  2. 激活环境：conda activate labelme
  3. 安装pyqt5 ：pip install pyqt5
  4. 安装labelme：pip install labelme
  5. 打开labelme：labelme

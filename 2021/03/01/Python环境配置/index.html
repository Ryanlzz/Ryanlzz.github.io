4.еҲӣе»әиҷҡжӢҹзҺҜеўғ

дёәд»Җд№ҲеҲӣе»әиҷҡжӢҹзҺҜеўғ?

иҷҡжӢҹзҺҜеўғе…¶е®һе°ұжҳҜPythonзҺҜеўғзҡ„дёҖдёӘеүҜжң¬пјҢеҸҜд»ҘдҝқиҜҒжҜҸдёӘйЎ№зӣ®йғҪжңүдёҖеҘ—зӢ¬з«Ӣзҡ„pythonиҝҗиЎҢзҺҜеўғпјҢеңЁдёҚеҗҢзҡ„иҷҡжӢҹзҺҜеўғдёӯеҸҜд»ҘйҖүжӢ©дёҚеҗҢзүҲжң¬зҡ„еә“гҖӮ

еҲӣе»әиҷҡжӢҹзҺҜеўғпјҢдё»иҰҒз”ЁдәҺTensorFlow

  1. еҲӣе»әиҷҡжӢҹзҺҜеўғпјҡconda create вҖ”name tensorflow-gpu python=3.8пјҢеҲӣе»әеңЁAnaconda3зҡ„envsдёӢгҖӮ

  2. жҝҖжҙ»зҺҜеўғпјҡconda activate tensorflow-gpu

  3. жҹҘзңӢзҺҜеўғпјҡconda info вҖ”envs

  4. е®үиЈ…TensorFlowпјҡpip install tensorflow-gpu

  5. жү“ејҖpythonпјҢиҫ“е…Ҙ import tensorflow as tf жҠҘеҰӮдёӢй”ҷиҜҜпјҡ

    Could not load dynamic library вҖҳcudart64_101.dllвҖҷ; dlerror: cudart64_101.dll not found

    еҺҹеӣ жҳҜжҲ‘иЈ…зҡ„TensorFlowзүҲжң¬жҳҜжңҖж–°зҡ„tensorflow-gpu 2.3.1пјҢдҪҶжҳҜе®ғиҝҳдёҚж”ҜжҢҒCUDA11.1пјҢеҠһжі•жҳҜдёӢиҪҪдҪҺзүҲжң¬зҡ„CUDAе’ҢcudnnгҖӮ

    tensorflowеҗ„дёӘзүҲжң¬йңҖиҰҒзҡ„CUDAзүҲжң¬д»ҘеҸҠCudnnзҡ„еҜ№еә”е…ізі»пјҡhttps://www.tensorflow.org/install/source_windows#install_gpu_support_optionalпјҢ

    д№ӢеҗҺжүҫеҲ°и§ЈеҶіеҠһжі•еҸӮиҖғпјҡhttps://blog.csdn.net/qq_32939413/article/details/105525025

    дёҚиҝҮжҲ‘д№ҹдёҚзқҖжҖҘз”ЁпјҢеҶҚзӯүзӯүTensorFlowжӣҙж–°еҗ§lolпјҢеҗ¬иҜҙPytorchж”ҜжҢҒDUDA11пјҢйӮЈз”ЁPytorchеҗ§гҖӮ

  6. йҖҖеҮәзҺҜеўғпјҡdeactivate

еҸӮиҖғпјҡ

  1. https://zhuanlan.zhihu.com/p/60647332
  2. https://blog.csdn.net/j879159541/article/details/93199150

еҲӣе»әиҷҡжӢҹзҺҜеўғпјҢдё»иҰҒз”ЁдәҺPytorch

  1. еҲӣе»әиҷҡжӢҹзҺҜеўғпјҡconda create вҖ”name pytorch python=3.8

  2. жҝҖжҙ»зҺҜеўғпјҡconda activate pytorch

  3. жҹҘзңӢзҺҜеўғпјҡconda info вҖ”envs

  4. е®үиЈ…Pytorchпјҡconda install pytorch torchvision torchaudio cudatoolkit=11.0 -c pytorchпјҢеҸӮиҖғе®ҳзҪ‘пјҡhttps://pytorch.org/пјҢиҰҒзӯүдёҖж®өж—¶й—ҙдәҶ......гҖӮ

  5. жү“ејҖpythonпјҢиҫ“е…Ҙ import torchпјҢж— жҠҘй”ҷпјҢжҲҗеҠҹпјҒ

  6. е®үиЈ…pandasпјҡconda install pandas

  7. е®үиЈ…Matplotlibпјҡconda install matplotlib

  8. е®үиЈ…scikit-imageпјҡconda install scikit-image

  9. е®үиЈ…sklearnпјҡconda install scikit-learn

  10. е®үиЈ…opencv-pythonпјҡpip install opencv-python -i https://pypi.mirrors.ustc.edu.cn/simple

    дёӢиҪҪеӨӘж…ўдәҶпјҢжҲ‘дҪҝз”ЁдәҶдёӯз§‘еӨ§зҡ„жәҗдёӢиҪҪпјҡ

    иҰҒжҚўе“ӘдёӘжәҗеҸӘйңҖиҰҒж”№-iеҗҺйқўзҡ„й“ҫжҺҘең°еқҖеҚіеҸҜ

    йҳҝйҮҢдә‘ http://mirrors.aliyun.com/pypi/simple/

    дёӯеӣҪз§‘жҠҖеӨ§еӯҰ https://pypi.mirrors.ustc.edu.cn/simple/

    иұҶз“Ј(douban) http://pypi.douban.com/simple/

    жё…еҚҺеӨ§еӯҰ https://pypi.tuna.tsinghua.edu.cn/simple/

    дёӯеӣҪз§‘еӯҰжҠҖжңҜеӨ§еӯҰ http://pypi.mirrors.ustc.edu.cn/simple/

  11. е®үиЈ…jupyter notebookпјҡconda install jupyter notebook

жіЁпјҡеҰӮжһңе®үиЈ…дёӯйҖ”е®үиЈ…еӨұиҙҘпјҢеҸҜд»ҘдҪҝз”Ёпјҡconda clean вҖ”allпјҢз„¶еҗҺйҮҚж–°е®үиЈ…пјҢдёҚз„¶еҸҜиғҪеҜјиҮҙCondaVerificationError

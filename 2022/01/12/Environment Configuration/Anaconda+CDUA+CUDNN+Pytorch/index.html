4.Create a virtual environment

Why create a virtual environment?

The virtual environment is actually a copy of the Python environment, which can ensure that each project has an independent Python running environment, and different versions of the library can be selected in different virtual environments.

Create a virtual environment Pytorch

  1. Create a virtual environment: conda create ‚Äìname pytorch python=3.8

  2. Activate the environment just created: conda activate pytorch

  3. Check environmental information: conda info ‚Äìenvs

  4. Install Pytorc: official website link: https://pytorch.org/
